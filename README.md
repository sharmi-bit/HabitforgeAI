# HabitForge AI 🚀

HabitForge AI is an AI-powered habit tracking and productivity app that helps users build better habits through personalized AI-generated plans, tracking, and insights.

## Features
- AI-generated habit plans  
- Habit tracking with streaks  
- Progress dashboard  
- Smart reminders and automation  
- User authentication  

## Tech Stack
- Next.js  
- TypeScript  
- Tailwind CSS  
- Supabase  
- Claude / Groq API  
- n8n  

## Goal
Make habit building simple, consistent, and AI-driven.
